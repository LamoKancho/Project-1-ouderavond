HTML

hier gaan wij u uitleggen waar we html voor gebruiken, hoe het werkt en wat het allemaal voor gebruikt word.


Wat is HTML en hoe werkt het? HTML staat voor Hypertext Markup Language, een programmeertaal die wordt gebruikt om webpagina's te schrijven. Hoewel HTML ook gedeeltelijk kan omgaan met design en styling, doen we dit meestal in een andere taal: CSS (Cascading Style Sheets). De structuur van HTML-pagina's zijn relatief gezien op een eenvoudige en gemakkelijke manier opgebouwd. HTML-pagina's beginnen met een DOCTYPE-declaratie. Deze verklaring vertelt de browser welke standaarden moeten worden geladen. Dan krijg je de tag die het HTML-document opent, gevolgd door de "head"-tag. In deze "head"-tag geven we de titel en link van de stylesheet door, enz. Net onder het "hoofd" bevindt zich het "lichaam". Neem de volledige structuur en inhoud van de pagina op in de "body". Binnen de “body” komt de gehele opbouw en inhoud van de pagina.




























































CSS

hier gaan wij u uitleggen waar we css voor gebruiken, wat het doet en hoe het ongeveer werkt


Cascading Style Sheets (kortweg CSS) is een programmeertaal die zorgt voor de opmaak van HTML-bestanden. Onderaan elke pagina van een website staat een HTML-bestand dat zorgt voor al uw zichtbare inhoud (tekst, afbeeldingen, links...). Nu denk je misschien, waarom heb ik CSS nodig? CSS zorgt ervoor dat je deze HTML-code kunt bewerken. Stel dat u een HTML-website hebt gemaakt zonder dat er een CSS-bestand is bijgevoegd. Hoe denk je dat het eruit ziet? Laten we eens kijken hoe het html-bestand eruit ziet zonder css en ermee. HTML-bestand zonder CSS

Als u geen stylesheet (css) definieert in uw html-bestand, gebruikt de browser een standaardstijl, genaamd "user-agent stylesheet". HTML-bestanden met CSS

Wanneer u een stylesheet (css) in uw html-bestand definieert, neemt de browser de controle over de stijl van het css-bestand dat aan dat html-bestand is gekoppeld. macbook met website met css
